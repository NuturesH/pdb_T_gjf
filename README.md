# pdb_T_gjf 
文件中包含一个类，给予这个类需要处理的文件名称，之后用两个函数进行处理。
  函数PDB_Coord第一步将vmd所生成的pdb文件转化为原子名称加坐标的形式。 
                第二步将所生成的坐标文件加上g16命令的模板文件生成初始的gjf文件
                第三步将所生成的gjf文件进行修改，修改其中chk文件的路径及名称
